WordPress Embedding Instructions

Method 1: iframe Embed (Recommended)

  1. Upload emoji-sentiment-embed.html to your WordPress Media Library
  2. Get the direct URL to the file
  3. Use this HTML in your WordPress editor:
<div style="text-align: center; margin: 20px 0;">
    <iframe 
        src="https://yoursite.com/wp-content/uploads/2025/01/emoji-sentiment-embed.html" 
        width="420" 
        height="620" 
        frameborder="0" 
        style="border-radius: 10px; box-shadow: 0 4px 20px rgba(0,0,0,0.1);">
    </iframe>
</div>

Method 2: WordPress Shortcode

Add this to your theme's functions.php file:

function emoji_sentiment_demo_shortcode() {
    $upload_dir = wp_upload_dir();
    $file_url = $upload_dir['baseurl'] . '/emoji-sentiment-embed.html';
    
    return '<div style="text-align: center; margin: 20px 0;">
        <iframe 
            src="' . esc_url($file_url) . '" 
            width="420" 
            height="620" 
            frameborder="0" 
            style="border-radius: 10px; box-shadow: 0 4px 20px rgba(0,0,0,0.1);">
        </iframe>
    </div>';
}
add_shortcode('emoji_sentiment_demo', 'emoji_sentiment_demo_shortcode');

Then use [emoji_sentiment_demo] in your posts/pages.

Method 3: Gutenberg Block

  1. In WordPress editor, add a "Custom HTML" block
  2. Paste the iframe code above
  3. Preview/publish your page

Method 4: Direct Hosting

  1. Upload emoji-sentiment-embed.html to your web server
  2. Access it directly: https://yoursite.com/emoji-sentiment-embed.html
  3. Share the direct link or embed via iframe

Centering & Responsive Design

Centering: The examples above include a wrapper div with text-align: center to center the demo on your page.

Mobile-responsive: For advanced responsive embedding, use this CSS:

<style>
.emoji-demo-container {
    width: 100%;
    max-width: 420px;
    margin: 0 auto;
}
.emoji-demo-container iframe {
    width: 100%;
    height: 620px;
    border: none;
    border-radius: 10px;
    box-shadow: 0 4px 20px rgba(0,0,0,0.1);
}
@media (max-width: 480px) {
    .emoji-demo-container iframe {
        height: 500px;
    }
}
</style>

<div class="emoji-demo-container">
    <iframe src="https://yoursite.com/wp-content/uploads/2025/01/emoji-sentiment-embed.html"></iframe>
</div>
